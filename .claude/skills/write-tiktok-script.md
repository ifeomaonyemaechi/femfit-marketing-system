# Skill: Write TikTok Script
# FemFit.fit Marketing Operations System

## WHAT THIS SKILL DOES
Generates a complete TikTok video script for FemFit.fit
including hook, talking points, shot list, text overlays,
and caption. Called by the social-media-manager agent.
Every script must feel like it was filmed by a real woman
who trains — not produced by a marketing team.

## WHEN TO USE THIS SKILL
Use when:
- A TikTok video needs a full production script
- Weekly content pack includes TikTok pieces
- A content calendar item is assigned to TikTok

## VIDEO FORMATS

### Talking Head (Direct to Camera)
Best for: Opinions, red flags, honest reviews, tips
Length: 30-60 seconds
Style: One-take energy, natural delivery, no teleprompter feel
Hook type: Bold statement or problem statement

### Training Footage
Best for: Product in action, workout demos, GRWM
Length: 30-45 seconds
Style: Real environment, natural lighting, no ring light
Hook type: Visual hook — show the action immediately

### List Format
Best for: Educational content, red flags, quick tips
Length: 30-45 seconds
Style: Fast cuts, text overlays drive the content
Hook type: Number hook — "4 things that..."

### POV Format
Best for: Home gym life, relatable moments
Length: 15-30 seconds
Style: Immersive, viewer is in the scene
Hook type: Situational — "POV: It's 5:47am..."

## HOOK RULES — CRITICAL
The first 3 seconds determine if anyone watches.

Good hooks:
- State a problem immediately: "Six months of leg days
  ruined my legging standards."
- Use a number: "Four sports bra red flags nobody
  warns you about."
- Make a bold claim: "Most squat-proof leggings aren't."
- Create curiosity: "We almost didn't make these."

Bad hooks — never use:
- "Hey guys, today I'm going to..."
- "So I wanted to share something with you..."
- Any greeting before the hook
- Starting with the brand name

## TIKTOK AESTHETIC RULES
- Natural lighting — no ring lights
- Real environments — home gym, actual gym, living room
- One-take energy — small stumbles are fine
- No overproduced cuts in talking head videos
- Text overlays should reinforce, not repeat speech
- Music low enough that voice is clear throughout

## EXECUTION INSTRUCTIONS
1. Confirm video format from brief
2. Write hook — test it before writing the rest
3. Write full script with talking points
4. Create shot list with camera directions
5. Write text overlay table with timing
6. Write audio direction notes
7. Write full caption separately
8. Check against brand voice rules
9. Save to outputs/FEMFIT_TIKTOK_[CONCEPT]_[YYYYMMDD].md

## QUALITY GATES
Before saving output, confirm:
[ ] Hook delivers tension or problem in under 3 seconds
[ ] No banned words in script or caption
[ ] Video length is 30-60 seconds maximum
[ ] Shot list has at least 5 shots
[ ] Text overlays table is complete with timing
[ ] Audio direction specifies music style and volume
[ ] Caption has hook, body, CTA, and hashtags
[ ] Aesthetic notes say no ring light
