# Skill: Write Blog Post
# FemFit.fit Marketing Operations System

## WHAT THIS SKILL DOES
Generates a complete SEO-optimised blog post for FemFit.fit.
Called by the seo-strategist agent. Every blog post must
rank for a target keyword AND be genuinely useful to the
reader. Traffic that doesn't convert is worthless.

## WHEN TO USE THIS SKILL
Use when:
- The content calendar includes a blog post
- A keyword gap needs to be filled
- A product needs educational support content
- A seasonal moment needs an SEO article

## BLOG POST TYPES

### How-To Guide
Purpose: Teach the reader how to do something specific
Target keywords: "how to choose gym leggings",
"how to test squat-proof leggings"
Length: 800-1,200 words
Structure: Problem → Steps → Product mention → CTA

### Product Education
Purpose: Help readers understand what makes quality gear
Target keywords: "best sports bra for HIIT",
"women's home gym equipment"
Length: 700-1,000 words
Structure: What to look for → What to avoid →
FemFit.fit solution → CTA

### Lifestyle and Training
Purpose: Reach women searching for training content
Target keywords: "home workout women", "training at home
with kids", "women's strength training beginners"
Length: 600-900 words
Structure: Relatable opening → Practical content →
Brand philosophy tie-in → CTA

### Comparison and Review
Purpose: Capture high-intent buyers comparing options
Target keywords: "squat proof leggings review",
"best gym leggings South Africa"
Length: 800-1,200 words
Structure: Criteria → Comparison → Recommendation → CTA

## POST STRUCTURE — ALWAYS FOLLOW

### Title
Format: [Primary keyword] — [Benefit or promise]
Character limit: Under 60 characters
Must include primary keyword naturally
Example: "How to Test Squat-Proof Leggings Before You Buy"

### Introduction (150 words)
Hook: One sentence that names the problem
Body: What the reader will learn and why it matters
No: "In this article we will discuss..."
Yes: Start with the problem or tension immediately

### H2 Sections (3-5 sections)
Each H2 answers one search intent question
Each section: 150-250 words
Include primary or secondary keyword in at least one H2
Use real, specific information — no filler

### Product Mention (1 per post maximum)
Natural integration only — never forced
Position as a solution to a problem raised in the post
Link anchor text should be descriptive, not "click here"

### Conclusion (100 words)
Summary of key points
One clear CTA — shop, read more, or sign up

### Meta Fields
Meta title: Under 60 characters, includes primary keyword
Meta description: Under 155 characters, includes keyword
and a benefit

## SEO RULES
- Primary keyword in: title, first 50 words, one H2,
  meta title, meta description
- Secondary keywords used naturally in body — never forced
- No keyword stuffing — if it sounds unnatural, remove it
- Internal links: mention one other relevant FemFit.fit
  page where relevant
- Image alt text: describe the image with keyword where
  natural

## FEMFIT.FIT BLOG VOICE
Same brand voice as all other content — direct, no fluff,
knowledgeable like a friend who trains.
Blog posts can be slightly longer in sentence structure
than social content but never corporate or academic.
Write like you are explaining something to a friend
who asked a genuine question.

## EXECUTION INSTRUCTIONS
1. Confirm post type and primary keyword from brief
2. Write title first — check character count
3. Write introduction — hook in first sentence
4. Write H2 sections in logical order
5. Insert one product mention naturally
6. Write conclusion with CTA
7. Write meta title and meta description
8. Check keyword placement
9. Check against brand voice rules
10. Save to outputs/FEMFIT_BLOG_[KEYWORD]_[YYYYMMDD].md

## QUALITY GATES
Before saving output, confirm:
[ ] Primary keyword in first 50 words
[ ] Primary keyword in at least one H2
[ ] Meta title under 60 characters
[ ] Meta description under 155 characters
[ ] Introduction starts with problem or hook
[ ] Only one product mention in the post
[ ] No banned brand voice words
[ ] Conclusion has one clear CTA
[ ] No keyword stuffing — all mentions feel natural
