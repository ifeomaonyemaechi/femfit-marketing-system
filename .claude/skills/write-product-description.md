# Skill: Write Product Description
# FemFit.fit Marketing Operations System

## WHAT THIS SKILL DOES
Generates SEO-optimised product descriptions for FemFit.fit
Shopify store. Called by the seo-strategist agent. Every
description must rank for a target keyword AND convert
browsers into buyers. Function first, aesthetics second.

## WHEN TO USE THIS SKILL
Use when:
- A new product is being added to the Shopify store
- An existing description is underperforming
- A product page needs SEO optimisation
- A collection page needs updated copy

## PRODUCT CATEGORIES

### Apparel (Leggings, Sports Bras, Tops)
Primary focus: Fit, function, and durability
Key concerns to address: Squat-proof, waistband stability,
support level, sizing accuracy, wash durability
Target keywords: "squat proof leggings women",
"high waist gym leggings", "sports bra medium support"

### Footwear (Training Shoes)
Primary focus: Support, grip, and training suitability
Key concerns to address: Arch support, stability,
sole grip, break-in period, training types suitable for
Target keywords: "women's training shoes",
"gym shoes women South Africa"

### Equipment (Mats, Kettlebells, Bands, Accessories)
Primary focus: Function, durability, and home gym suitability
Key concerns to address: Size, weight, grip, storage,
suitability for specific exercises
Target keywords: "home gym equipment women",
"kettlebell women", "non slip yoga mat"

## DESCRIPTION STRUCTURE — ALWAYS FOLLOW THIS ORDER

### 1. HOOK (1 sentence)
What problem does this product solve?
Must include primary keyword naturally.
Must lead with function, not aesthetics.

Good: "Leggings built to stay put through every squat,
deadlift, and hip thrust — no rolling, no sliding,
no adjusting between sets."
Bad: "Beautiful high-waisted leggings in premium fabric
that will make you feel amazing at the gym."

### 2. KEY FEATURES (3 bullet points)
Each bullet: Feature name + specific detail
Format: **[Feature name]** — [specific explanation]

Good: "**Double-layer waistband** — stays put through
100+ squats without rolling or sliding"
Bad: "**Premium waistband** — for ultimate support
and comfort"

Rules:
- Every claim must be specific and provable
- No superlatives — "best", "ultimate", "perfect"
- Technical details welcome — fabric composition,
  weight, dimensions where relevant

### 3. FIT AND FEEL (1-2 sentences)
How does it move, feel, and perform during training?
Write from experience — not from a product spec sheet.

Good: "The fabric has enough compression to hold
everything in place without restricting movement
through full depth squats or Romanian deadlifts."
Bad: "Comfortable and flexible for all your workouts."

### 4. SIZING NOTE (1 sentence)
Address the sizing concern directly.
Link to size guide if available.

Good: "Runs true to size — if you're between sizes,
size up for comfort during heavy lifts."

### 5. CARE INSTRUCTIONS (1 sentence)
Cold wash, air dry. Skip fabric softener.
Specific to the product material.

### 6. CTA (1 sentence)
Direct purchase prompt.
Not "add to cart" — something specific to the product.

Good: "If your current leggings are letting you down
mid-session, these won't."

## SEO RULES FOR PRODUCT DESCRIPTIONS
- Primary keyword in first sentence naturally
- Secondary keyword in features or fit section
- Word count: 150-200 words per description
- No keyword stuffing — if it sounds forced, remove it
- Meta title: under 60 characters, includes keyword
- Meta description: under 155 characters, benefit-led

## SHOPIFY FORMATTING
- Use **bold** for feature names in bullet points
- Short paragraphs — 1-3 sentences each
- Bullet points for features section only
- No headers inside the description itself
- HTML formatting if brief specifies

## EXECUTION INSTRUCTIONS
1. Confirm product category and name from brief
2. Identify primary keyword
3. Write hook with keyword in first sentence
4. Write 3 feature bullets with specific details
5. Write fit and feel section
6. Write sizing note
7. Write care instructions
8. Write CTA
9. Write meta title and meta description
10. Check keyword placement and character counts
11. Check against brand voice rules
12. Save to outputs/FEMFIT_PRODUCTDESC_[PRODUCT]_[YYYYMMDD].md
